## 环境依赖

### 爬虫

1. Python：3.X
2. 所需库：
   - dateutil
     - 安装方法：`pip install python-dateutil`
   - selenium
     - 安装方法：`pip install selenium`
3. 模拟驱动：chromedriver或edgedriver或firfoxdriver等。到官网下载与浏览器对应的版本，放在Python安装路径下的Scripts目录下
